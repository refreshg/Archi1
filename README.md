# Bitrix24 Sales Report (Archi)

Bitrix24 სეილსის რეპორტი – Pipeline 0.

## ფუნქციები

- თარიღების არჩევა მომხმარებლის მიერ
- ფეიჯინგი (10, 25, 50, 100 ჩანაწერი გვერდზე)
- Excel ექსპორტი (.xlsx)
- დილების ცხრილი – დაკლიკება გადახვიდე Bitrix24-ში

## ლოკალური გაშვება

```bash
npm install
npm start
```

გახსენი: http://localhost:3000

## Vercel-ზე განხორციელება

1. [vercel.com](https://vercel.com)-ზე შედი და გაიარე ავტორიზაცია
2. **Add New Project** → **Import Git Repository**
3. აირჩიე: **refreshg/Archi1**
4. **Deploy** დააჭირე

ან გამოიყენე CLI:

```bash
npm i -g vercel
vercel login
vercel
```
